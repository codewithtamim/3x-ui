> **This is an enhanced fork of [MHSanaei/3x-ui](https://github.com/MHSanaei/3x-ui)** with additional features such as API key management and Swagger API documentation.

**3X-UI** — یک پنل کنترل پیشرفته مبتنی بر وب با کد باز که برای مدیریت سرور Xray-core طراحی شده است. این پنل یک رابط کاربری آسان برای پیکربندی و نظارت بر پروتکل‌های مختلف VPN و پراکسی ارائه می‌دهد.

> [!IMPORTANT]
> این پروژه فقط برای استفاده شخصی و ارتباطات است، لطفاً از آن برای اهداف غیرقانونی استفاده نکنید، لطفاً از آن در محیط تولید استفاده نکنید.

## شروع سریع

```
bash <(curl -Ls https://raw.githubusercontent.com/codewithtamim/3x-ui/main/install.sh)
```

برای مستندات کامل، لطفاً به [ویکی پروژه](https://github.com/codewithtamim/3x-ui/wiki) مراجعه کنید.

## تقدیر

این پروژه یک فورک از [MHSanaei/3x-ui](https://github.com/MHSanaei/3x-ui) است. تشکر ویژه از:

- [MHSanaei](https://github.com/MHSanaei) — سازنده و نگهدارنده اصلی 3X-UI
- [alireza0](https://github.com/alireza0/) — مشارکت‌کننده اصلی پروژه اصلی
